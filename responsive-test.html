Manual Testing Instructions

üîç What to Look For:

  1. Layout Breaks: Content overlapping or cut off
  2. Navigation Issues: Menu not accessible on mobile
  3. Button Sizes: Too small to tap comfortably
  4. Text Readability: Too small or hard to read
  5. Form Problems: Inputs too small or hard to use
  6. Image Issues: Not scaling properly
  7. Horizontal Scrolling: Should be avoided
  8. Modal Behavior: Should fit screen properly

üß™ Testing Steps:

  1. Start with desktop view and gradually reduce screen size
  2. Test each major breakpoint (320px, 768px, 1024px, 1440px)
  3. Try all major features at each size
  4. Test both portrait and landscape orientations
  5. Verify touch interactions work on mobile
