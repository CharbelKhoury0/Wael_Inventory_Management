Manual Testing Instructions

To test CRUD operations in the CMS Pro application:

  1. Create (Add Item):
    • Click the "Add Item" button in the Items page
    • Fill out the form with test data
    • Submit and verify the item appears in the list
  2. Read (View Item):
    • Click the eye icon on any item to view details
    • Verify all information displays correctly
  3. Update (Edit Item):
    • Click the edit icon on any item
    • Modify some fields and save
    • Verify changes are reflected in the list
  4. Delete (Remove Item):
    • Click the delete icon on any item
    • Confirm deletion in the modal
    • Verify item moves to trash
    • Test undo functionality
